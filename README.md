#Online Task Manager (Full-Stack Web App with Login Feature)

 Project Description

This is a simple full-stack Task Manager (To-Do App) project demonstrating frontend and backend skills.

 Users can:
- Login with username and password.
- Add, view, and manage tasks.
- Tasks are stored in a persistent SQLite database.

---

Technologies Used

| Layer     | Technology            |
|-----------|-----------------------|
| Frontend  | React.js (HTML, CSS, JavaScript) |
| Backend   | Node.js + Express.js + CORS |
| Database  | Python + SQLite       |

---

Key Features

✔️ Simple Login Page (Username: `admin`, Password: `1234`)  
✔️ Add New Tasks  
✔️ View All Tasks  
✔️ Data Persistence with SQLite  
✔️ CORS Configuration for API Access
